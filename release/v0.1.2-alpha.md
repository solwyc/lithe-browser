Lithe 0.1.2-alpha is a portable Windows x86-64 preview focused on modern web compatibility without losing Lithe's lightweight, private-by-default philosophy.

What's new:

- A persistent retro-pixel Vibes control strip with Back, dislike, like, and Next actions.
- Local likes and dislikes now update the on-device recommendation profile without sending browsing history to Lithe.
- A fully rebranded About panel with Lithe project, privacy, source, release, and issue links.
- Original cream-and-mint Lithe bunny artwork across About, onboarding, and Windows browser surfaces.
- DuckDuckGo remains the default search engine, with telemetry disabled and AI Protection enabled by default.
- Media and compatibility checks cover YouTube playback plus H.264, VP9, VP8, AV1, HEVC, AAC, and MP3 capability.

This is an unsigned, experimental portable alpha. Extract the entire ZIP before running `lithe.exe`; the executable cannot run by itself. Verify the accompanying SHA-256 checksum. This build does not yet have a production automatic-update channel and should not be treated as a replacement for a browser receiving production security updates.

Vibes classification and the taste profile stay local. When live discovery is enabled, Lithe sends generic category searches to DuckDuckGo and may pre-read up to three candidate pages without cookies, scripts, or referrers.
