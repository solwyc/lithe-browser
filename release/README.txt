Lithe Browser 0.1.0-alpha for Windows x86-64
================================================

This is an experimental, unsigned portable alpha build.

1. Extract the entire ZIP to a folder.
2. Run lithe.exe from inside that folder.
3. Keep the other extracted files beside lithe.exe; the executable does not
   work by itself.

Windows SmartScreen may show an unrecognized-app warning because this build is
not code-signed. Verify the SHA-256 checksum against the checksum published on
the GitHub release before running it. Do not bypass a warning for a file whose
checksum does not match.

This alpha does not yet have a production automatic-update channel. It should
not be treated as a replacement for a browser receiving production security
updates.

Vibes classification runs locally. Vibes discovery, when enabled, sends only
generic category searches to DuckDuckGo and may pre-read up to three candidate
pages without cookies, scripts, or referrers.

Project and source: https://github.com/solwyc/lithe-browser
Release source: https://github.com/solwyc/lithe-browser/tree/v0.1.0-alpha
